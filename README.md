# Student Performance Management System (C++)

## Features
- Add, View, Search, Update, Delete Students
- File Handling (Data persistence using text file)
- Subject-wise marks tracking (Maths, Science, English)
- Automatic average and grade calculation
- Performance analysis (Topper, Top 3, Below average)
- Subject-wise topper system

## Tech Used
- C++
- File Handling
- Vectors
- OOP concepts (struct based design)

## Learning Outcome
- Data structures usage
- File handling in C++
- Modular programming
- Real-world system design
